## ğŸ›  Compilation Details

* **C/C++**: Uses **Emscripten (emcc)**. Generates an HTML/JS wrapper by default to support standard libraries (`stdio.h`, `iostream`).
* **C#**: Uses **.NET WASM Tools**. Compiles your project into a browser-compatible WASM runtime.
* **Runner**: The `wwwroot/index.html` in the root acts as a "Virtual Machine." It uses **JSZip** to extract and execute your builds directly in the browser memory without a backend server.
